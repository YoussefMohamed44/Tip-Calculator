# Tip Calculator

A modern Android tip calculator app built with Jetpack Compose.

## Features

- 💰 Calculate tips based on bill amount and service quality
- 🔄 Round up tip option
- 🎯 Quick tip percentage buttons (10%, 15%, 20%)
- 🌙 Dark mode support
- ✨ Material Design 3 UI
- 🧹 Clear button to reset calculations
- ✅ Input validation with error messages

## Technologies Used

- Kotlin
- Jetpack Compose
- Material Design 3
- Android Studio

## How to Run

1. Clone this repository
2. Open in Android Studio
3. Run on an emulator or physical device

## Installation

```bash
git clone https://github.com/YoussefMohamed44/Tip-Calculator.git
```

## Usage

1. Enter the bill amount
2. Enter the tip percentage or use quick buttons
3. Toggle "Round Up" if you want to round the tip
4. View the calculated tip amount
5. Use "Clear" to start a new calculation
